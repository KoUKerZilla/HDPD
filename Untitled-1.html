HDPD 8.12

Quickly and easily tool to analize and make prediction with Near Infrared spectra. This tool is actually in alpha version, contact us for more informations and/or send us feedback

How ? :

  • The models in the tool come from an ASD LabSpec 4 Standard-Res Lab Analyzer
  • This tool uses NIRS spectra in ASCII format
  • Please download only files in .txt format
  • Do you have any doubts? Use our  test file
